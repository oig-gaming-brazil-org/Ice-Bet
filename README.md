# Ice.Bet - Excelência em Cassino Online

Repositório de infraestrutura de TI e compliance da **[Ice.Bet](https://ice.bet.br)**. Uma marca de alta performance operada pela OIG GAMING BRAZIL LTDA, sob a governança do One Internet Group.

## Sobre a Plataforma
A **Ice.Bet** é a nossa aposta focada na excelência absoluta em **cassino online** e estabilidade transacional. Construída para suportar altíssimos volumes de acessos no mercado de **iGaming**, a plataforma utiliza segurança criptográfica avançada para proteger os dados e os fundos dos jogadores em todos os nossos **jogos online**.

## Estabilidade Operacional
Nossa engenharia garante resiliência contra ataques DDoS e redundância de servidores, proporcionando uma experiência de jogo ininterrupta e premium.
